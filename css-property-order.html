Footnote

  • 誤植っぽい箇所を修正した。
  • プロパティの記述順はCSS2 Specificationで出てくる順序ですに厳格に沿うべく shorthand property を各プロパティグループの最後にした (一部 specification に沿って shorthand property が最初)。
  • CSS 3 はモジュールをアルファベット順に列挙。 各モジュールの specification 出現順序で、標準プロパティに続いて対応するベンダ拡張プロパティを -moz-, -ms-, -o-, -webkit- の順で追加。
  • CSS 3 の網羅性はかなりテキトー。
  • プロパティに出典のリンクを追加した。
