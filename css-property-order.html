Footnote

  • 誤植っぽい箇所を修正した。
  • プロパティの記述順はCSS2 Specificationで出てくる順序ですに厳格に沿うべく shorthand property を各プロパティグループの最後にした (outline のみ specification に沿って shorthand property が最初)。
  • CSS 2.1 準拠を再確認した。加えて CSS3 Playground by Mike Plate で使用できる CSS 3 プロパティを各モジュール (アルファベット順) ごとの出現順序で追加した。
  • 各 UserAgent 独自プロパティも一部追加した。-moz, -o, -webkit のアルファベット順で最後尾に追加し、CSS 3 Specification の出現順序による。
  • 各プロパティに出典のリンクを追加した。
